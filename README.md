# 게임 소개
\<Crashy Road>(2023)은 2022년에 출시한 동명의 게임 리메이크 버전으로, 성공회대학교 인디 게임 개발팀 ‘팀 아이스크림 햄버거’에서 제작한 미니 아케이드 레이싱 게임입니다.

플레이어는 사방에서 몰려오는 경찰들과 추격전을 벌이며 드넓은 도심 속에서 아슬아슬한 레이싱을 펼치게 됩니다. 

자신만의 곡예 운전 스킬을 익히며 경찰들을 따돌리세요.

무작위로 등장하는 아이템을 노리고, 멈출 줄 모르는 자동차와 함께 최대한 오랫동안 살아남으세요. 

목표는 하이 스코어입니다. 과연 과거의 나를 이길 수 있을까요?

<br/>

# 게임 특징
**1. 경찰과의 스릴 있는 추격전**
   
플레이어는 게임을 시작함과 동시에 경찰의 추격을 받게 됩니다. 

이유는… 글쎄요. 각자의 상상에 맡기죠.

사방에서 몰려드는 경찰들은 A* 알고리즘을 기반으로 플레이어까지의 최단 거리를 탐색합니다. 

코너를 만나면 사고를 막기 위해 속도를 줄이고, 플레이어가 차도를 벗어나면 사전에 베이크(Bake) 된 지형 정보에 따라 코스트(Cost)를 계산하여 차도로 갈 것인지, 인도를 거슬러서 갈 것인지를 결정합니다. 

플레이어는 경찰들을 따돌리기 위해 급커브를 연달아 선보이거나, 바다에 빠지기 직전에 핸들을 꺾어 경찰차들을 함정에 빠트리는 등 다양한 곡예 운전 기술을 선보여야 합니다. 

무작위로 나타나는 아이템은 플레이어에게 제한 시간 동안 특수 능력을 부여함으로써 이어지는 추격전에 변수를 더해줍니다.

<br/>

**2. 높은 게임 난이도, 낮은 진입 장벽으로 짧고 굵게 반복적으로 즐기는 미니 게임**

게임의 난이도는 상당히 높은 편입니다. 

아차! 하는 순간에 어디선가 달려드는 경찰차, 도심을 이루는 장애물과의 접촉 사고는 곧장 게임 오버로 이어집니다. 

즉, 게임 한판은 짧고 굵게 설계되어 있습니다. 

플레이어는 이전 판보다 더 오래 살아남겠다는 다짐을 반복하며 게임 오버와 다시 시작을 반복하게 됩니다. 

이런 모습을 연출하기 위해서는 게임을 학습하는 데 오랜 시간이 걸리면 안 됩니다. 

\<Crashy Road>는 A키와 D키만 다룰 수 있으면 게임을 플레이할 수 있습니다. 

학습은 쉽고 간단하지만, 게임의 난이도는 높게 설정해 반복 플레이를 해도 쉽게 지루해지지 않도록 설계했습니다.

<br/>

**3. 로우 폴리곤 스타일의 미려한 비주얼 퀄리티**

게임의 아트 요소들은 로우 폴리곤 스타일의 3D 그래픽을 사용합니다. 

파스텔로 칠한 것 같은 깔끔한 비주얼, 장난감 자동차를 운전하는 듯한 아기자기한 모습들은 현실적이지 않지만 가볍게 즐길 수 있는 아케이드 레이싱 게임의 분위기를 잘 표현하고 있습니다. 

머티리얼 디자인, 파스텔 색조로 맞춰진 UI 디자인도 그러한 분위기에 잘 녹아듭니다. 

게임 오버가 될 때마다 ‘팡팡 터지는’ 느낌을 주는 중독성 있는 SFX(사운드 이펙트)와 VFX(비주얼 이펙트)는 플레이어들이 게임에서 패배하더라도 즐거움을 느끼게 유도합니다.
