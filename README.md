# 개요

Team ICH의 2번째 작품, Crashy Road (2023)은 2022년 출시한 Crashy Road (2022)의 리메이크 작품입니다.

PAKO: Car chase simulator를 모티브로 한 이번 작품은 경찰의 추격을 피하며 하이 스코어를 노리는 아케이드 미니 레이싱 게임입니다.

넓은 도심에서 펼쳐지는 긴박한 경찰 추격전이 이번 작품의 핵심으로, 경찰과 장애물을 피하며 최대한 오래 살아남아야 합니다.

도심 곳곳에 등장하는 아이템을 활용해 도로 위를 엉망진창으로 만들어 보세요! 

<br/>

# 더욱 향상된 그래픽

이전 작품에서 지적된 요소 중 하나였던 어색한 라이팅 설정을 대폭 개선하여 더욱 몰입감 높은 그래픽을 구현하였습니다.

라이트 맵을 활용해 건물이나 도심 속 장애물들의 그림자를 베이킹하여 표현하였고, 플레이어의 차량과 경찰차의 경광등처럼 동적 요소들에는 실시간 라이팅을 적용하였습니다.

또한 이전보다 더 다양해진 VFX를 적용하여 게임의 사실감과 재미를 더했습니다.

<br/>

# '자동차'스러운 조작감 개선

빙판길 위를 미끄러지듯이 운전하는 듯이 느껴졌던 이전 작품의 조작감을 대폭 개선하여 아케이드 미니 레이싱 게임에 어울리는 조작감을 구현하였습니다.

4개의 Wheel Collider를 사용했던 이전 작품과 달리, 이번에는 자동차 몸체를 감지하는 Box Collider와 바퀴의 역할을 담당하는 커다란 Sphere Collider 하나를 힌지로 고정하는 방법을 적용하였습니다.

이를 통해 바퀴의 크기가 너무 작아 제대로 된 물리 효과를 보여주지 못했던 한계를 극복했으며, 마찰 계수, 자동차의 무게 등을 손쉽게 조절할 수 있도록 하여 아케이드 게임 속 자동차에 최적화된 조작감을 구현하였습니다.

<br/>

# 박진감 있는 경찰 추격전

맵 곳곳에서 나타나 플레이어를 추격하는 경찰들을 따돌리며 최대한 오래 살아남는 게임의 핵심 요소는 PAKO: Car chase simulator를 모티브로 하고 있습니다.

Crashy Road (2023)은 모티브가 된 PAKO: Car chase simulator보다 더욱 넓은 도심을 배경으로 하고 있으며, 아이템의 요소를 간략화하여 레이싱에 집중할 수 있도록 유도했습니다.

또한 맵 곳곳에 있는 가로등, 나무 등 규모가 작은 장애물들은 플레이어가 치고 나갈 수 있는 장애물로 배치하여 작품의 이름에 걸맞게 도로 위를 '엉망진창'으로 만들며 레이싱을 즐길 수 있게 하였습니다.

경찰차의 경우, Unity 엔진의 NavMeshAgent는 사실적인 물리 엔진을 구현할 수 없다는 한계를 극복하고자, NaveMesh를 기반으로 경로를 탐색한 후, 일정 속도를 넘어서면 코너에 가까워졌을 때 브레이크를 잡아 속도를 줄이는 방식을 통해 자동차의 사실적인 움직임과 A* 알고리즘 기반의 효율적인 길 찾기 성능을 모두 활용하였습니다.

<br/>

# 게임에 다채로움을 더하는 VFX/SFX

이전 작품보다 더 화려한 VFX를 적용하여 아이템을 먹었을 때의 효과를 텍스트 없이도 직관적으로 이해할 수 있게 유도했으며, 사고가 발생했을 때도 '정신없이 펑펑 터지는' 느낌을 받을 수 있게 하였습니다.

게임의 '긴박한 추격전' 분위기에 어울리도록 박진감 있는 BGM과 SFX를 활용해 음향 효과도 대폭 개선했습니다.

<br/>

# 게임의 몰입감을 높여주는 파스텔 스타일의 UI

'로우 폴리곤' 스타일의 게임 그래픽과 어울리도록 통일감 있는 파스텔 느낌의 색상과 직사각형 형태의 단순하고 깔끔한 UI를 적용했습니다.

은은하게 흐르는 메인 BGM과 도심을 천천히 비행하는 배경 애니메이션을 통해 메인 화면에서는 잔잔한 느낌을 주었으며, 다양한 설정 기능을 제공해 게임의 편의성도 강화하였습니다.

게임을 시작할 때, 자동으로 표시되는 튜토리얼 화면을 통해 갑자기 게임이 시작되어 플레이어가 당황하는 일이 없도록 하였으며, 별도의 메뉴를 조작하지 않아도 자연스럽게 게임의 규칙을 이해할 수 있도록 만들었습니다.

또한 '경기천년체'를 적용해 게임 속 텍스트의 가독성을 높였으며, 게임을 플레이하는 동안에는 텍스트를 최소화하여 게임에 더욱 몰입할 수 있도록 유도했습니다.

# Team ICECREAM HAMBURGER는...

팀 아이스크림 햄버거(Team ICECREAM HAMBURGER), 이하 Team ICH는 성공회대학교 학생들이 모여 창설한 대한민국의 인디 게임 개발팀입니다.

"놀이는 인간을 만들며, 인간은 놀이를 만든다."라는 모토 아래 일상에서 마주치는 수많은 경험에서 얻은 다양한 아이디어를 다채롭게 섞고, 바꾸고, 더하면서 색다르고 독특한 게임을 개발하기 위해 노력하고 있습니다.

다양한 경험, 다양한 시행착오, 다양한 발상과 상상력으로 단순히 '놀이를 즐기는 사람'을 넘어 '놀이를 만드는 사람'으로 나아가기 위해 오늘도 두근거리는 마음으로 키보드를 두들기고, 그림을 그리고, 글을 써 내려가고 있습니다.

Team ICH에서 개발하는 모든 게임은 Github를 통해 오픈 소스로 제공되며, 무료로 출시/배포됩니다.

Github: https://github.com/Team-IceCreamHamburger
